# Reload All Tabs

A minimal Chrome extension that reloads all tabs in the current window with one click or keyboard shortcut.

## Features

- Click the extension icon to reload all tabs instantly
- Keyboard shortcut support (default: `Cmd+Shift+E` on Mac, `Ctrl+Shift+E` on Windows/Linux)
- Only affects the current window, not all browser windows

## Installation

### From source

1. Clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (toggle in top right)
4. Click **Load unpacked**
5. Select the cloned folder

### Customize keyboard shortcut

1. Go to `chrome://extensions/shortcuts`
2. Find "Reload All Tabs"
3. Set your preferred key combination

## Why?

Chrome doesn't have a native "reload all tabs" hotkey. The manual method requires shift-clicking to select all tabs, then pressing refresh. This extension makes it one action.
